Project Description:

Audio Input host demonstration application. This gives a simple reference application for implementing a USB Audio host, for USB Audio devices using the USB Audio 1.0 class profile.

Incoming audio will output in 8-bit PWM onto the timer 3 output compare channel A, and will also be indicated on the board LEDs. Decouple the PWM output with a capacitor and attach to a speaker to hear the audio.
